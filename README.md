# 🐾 PAWSAFE — Animal Charity Website

A modern, responsive animal charity website built using **HTML5, CSS3, and Vanilla JavaScript**. This project was developed as **Project 3** for the **HexSoftwares Frontend Development Internship** and later polished as part of the **Fazal Labs** portfolio ecosystem.

# 📖 Overview

PAWSAFE is a fictional animal rescue and adoption organization designed to raise awareness about animal welfare while demonstrating modern frontend web development techniques.

The website showcases rescue programs, adoption opportunities, volunteer registration, donation impact, and responsive user interface design through a clean and engaging single-page experience.

---

# 🌍 Mission

PAWSAFE is dedicated to rescuing, rehabilitating, and rehoming stray and abandoned animals while encouraging responsible pet ownership and community involvement.

# 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/fazal305/HexSoftwares_AnimalCharityWebsite.git
```

Open the project:

```bash
cd HexSoftwares_AnimalCharityWebsite
```

Open **index.html** in your browser or use **VS Code Live Server**.

No build tools or package installation are required.

---

# 🏗 Architecture Notes

The project follows a simple static frontend architecture.

* **index.html** contains the website structure and content.
* **styles.css** manages layout, responsive design, animations, and visual styling.
* **script.js** controls interactive features including filtering, counters, scrolling, forms, and UI behavior.
