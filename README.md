# 🐾 PAWSAFE - Animal Charity Website

A responsive animal charity website built for the HexSoftwares Internship Project 3.

## 🌍 Mission
PAWSAFE is dedicated to rescuing, rehabilitating, and rehoming stray and abandoned animals in Karachi, Pakistan.

---

## 🚀 Features

- 🐶 Animal adoption filter system
- 💰 Interactive donation system with impact calculation
- 🤝 Volunteer application form with validation
- 📞 Contact form with conditional fields
- 📊 Animated counters (rescued animals, adoptions, volunteers)
- 📱 Fully responsive design
- ✨ Smooth scroll & scroll animations
- 🔝 Back to top button
- 🧠 Vanilla JavaScript interactivity (no frameworks)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
